### CGI

通用网关接口(CGI) 是一个Internet标准，允许Web服务器运行一个服务器端程序，称为CGI脚本

一般的，CGI脚本都放在一个名为cgi-bin的特殊文件夹下，这样Web服务器才能知道在哪里查找CGI脚本

事实上，所有Web应用都要在Web服务器上运行，  

Python的Web服务器包含在http.server库模块中
